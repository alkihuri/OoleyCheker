# OoleyCheker

Серверное приложение для интеграции со СКУД «ЭРА 5000», предназначенное для мониторинга посещений коворкинга и автоматической отправки уведомлений в Telegram.

## Описание

OoleyCheker — это десктопное приложение (Windows Forms, .NET Framework 4.6.1), которое подключается к базе данных системы контроля и управления доступом (СКУД) «ЭРА 5000» и в режиме реального времени отслеживает события прохода пользователей.

### Основные функции

- **Подключение к СКУД «ЭРА 5000»** — приложение подключается к базе данных Firebird (CBASE.FDB), используемой СКУД, и считывает события прохода из таблиц `FB_EVN` и `FB_USR`.
- **Мониторинг посещений коворкинга** — отслеживает, кто из пользователей прошёл через турникет, на какой этаж он попал (5-й или 6-й этаж).
- **Уведомления в Telegram-канал** — при фиксации нового прохода отправляет сообщение в Telegram-группу с информацией о пользователе и этаже посещения.
- **Персональные уведомления** — по количеству посещений отправляет личное уведомление пользователю о том, сколько у него осталось визитов.
- **Непрерывный мониторинг** — фоновый процесс (BackgroundWorker) постоянно опрашивает базу данных, обнаруживая новые события.

## Как это работает

1. Пользователь запускает приложение и нажимает кнопку **«Connect 2 DB»** для подключения к базе данных СКУД «ЭРА 5000».
2. После подключения нажимает **«Start»** — запускается фоновый процесс мониторинга.
3. Приложение непрерывно опрашивает базу данных на наличие новых событий прохода.
4. При обнаружении нового события:
   - Определяется пользователь и этаж (по идентификатору считывателя).
   - Информация отображается в интерфейсе приложения.
   - Если включена опция **«Telegram»**, уведомление отправляется в Telegram-группу.

## Требования

- Windows OS
- .NET Framework 4.6.1 или выше
- Установленный сервер Firebird (СКУД «ЭРА 5000»)
